API Endpoint

Make a GET request to download any YouTube video. 

GET https://your-api.onrender.com/download?url=YOUTUBE_VIDEO_URL

Parameters:

  • url (required) - YouTube video URL

Usage Examples 

curl "https://your-api.onrender.com/download?url=https://www.youtube.com/watch?v=dQw4w9WgXcQ"
const response = await fetch('https://your-api.onrender.com/download?url=https://www.youtube.com/watch?v=dQw4w9WgXcQ');
const data = await response.json();
console.log(data);
import requests

url = "https://your-api.onrender.com/download"
params = {"url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"}

response = requests.get(url, params=params)
data = response.json()
print(data)

API Response

The API returns a JSON response with download information. 

{
  "message": "Downloaded successfully!",
  "file_saved_to": "/tmp/abc123.mp4",
  "title": "Rick Astley - Never Gonna Give You Up",
  "duration": 213,
  "file_size": "3.2 MB"
}

Response Fields:

  • message - Status message
  • file_saved_to - Server file path
  • title - Video title
  • duration - Duration in seconds
  • file_size - File size
