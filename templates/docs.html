API Endpoint

Make a simple GET request to download any YouTube video. Our API processes the video using yt-dlp and returns comprehensive download information. 

GET https://your-api.onrender.com/download?url=YOUTUBE_VIDEO_URL

Query Parameters:

url REQUIRED The YouTube video URL to download

const downloadVideo = async (youtubeUrl) => {
  try {
    const response = await fetch(`https://your-api.onrender.com/download?url=${encodeURIComponent(youtubeUrl)}`);
    const data = await response.json();
    
    if (response.ok) {
      console.log('Download successful:', data);
      return data;
    } else {
      throw new Error(data.error || 'Download failed');
    }
  } catch (error) {
    console.error('Error:', error.message);
    throw error;
  }
};

import requests
import json

def download_video(youtube_url):
    """Download a YouTube video using the API"""
    api_url = "https://your-api.onrender.com/download"
    params = {"url": youtube_url}
    
    try:
        response = requests.get(api_url, params=params, timeout=30)
        response.raise_for_status()
        
        data = response.json()
        print(f"Download successful: {data}")
        return data
        
    except requests.exceptions.RequestException as e:
        print(f"Error: {e}")
        return None

# Usage
result = download_video("https://www.youtube.com/watch?v=dQw4w9WgXcQ")
if result:
    print(f"Video saved to: {result.get('file_saved_to')}")
const axios = require('axios');

const downloadVideo = async (youtubeUrl) => {
  try {
    const response = await axios.get('https://your-api.onrender.com/download', {
      params: { url: youtubeUrl },
      timeout: 30000 // 30 seconds timeout
    });
    
    console.log('Download successful:', response.data);
    return response.data;
    
  } catch (error) {
    if (error.response) {
      console.error('API Error:', error.response.data);
    } else {
      console.error('Network Error:', error.message);
    }
    throw error;
  }
};

// Usage with async/await
(async () => {
  try {
    const result = await downloadVideo('https://www.youtube.com/watch?v=dQw4w9WgXcQ');
    console.log(`Video saved to: ${result.file_saved_to}`);
  } catch (error) {
    console.error('Failed to download video');
  }
})();

API Response Format

The API returns a comprehensive JSON response with download information, video metadata, and file details.

Success Response (200 OK) 

{
  "success": true,
  "message": "Downloaded successfully!",
  "data": {
    "file_saved_to": "/tmp/abc123.mp4",
    "title": "Rick Astley - Never Gonna Give You Up (Official Video)",
    "duration": 213,
    "file_size": "3.2 MB",
    "format": "mp4",
    "quality": "720p",
    "download_time": 2.5
  }
}

Error Response (400/500) 

{
  "success": false,
  "error": "Invalid YouTube URL provided",
  "code": 400
}

Response Fields:

success - Boolean indicating if the request was successful
message - Human-readable status message
data.file_saved_to - Server file path where video is saved
data.title - Original video title from YouTube
data.duration - Video duration in seconds
data.file_size - Downloaded file size
data.format - Video format (mp4, webm, etc.)
data.quality - Video quality (720p, 1080p, etc.)
